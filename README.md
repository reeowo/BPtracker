# BPTracker — 血压记录 App

一款简洁的 iOS 血压管理应用，帮助用户记录、查看和分析血压及心率数据。

## 功能特性

- **今日总览**：展示当天所有测量数据的平均值，自动判断血压等级并给出颜色标识
- **快速记录**：一键打开输入面板，录入收缩压、舒张压、心率及备注
- **历史记录**：按天分组展示所有历史数据，支持左滑删除单条记录
- **统计趋势**：交互式折线图展示 7/14/30 天的血压和心率趋势，支持触摸查看单点数值
- **周期概览**：收缩压、舒张压、心率的范围统计及测量次数汇总
- **设置**：查看数据统计信息、App 版本，支持一键清除所有记录

## 血压等级分类

依据中国高血压指南：

| 等级        | 收缩压 | 舒张压 | 颜色 |
| ----------- | ------ | ------ | ---- |
| 偏低        | < 90   | < 60   | 蓝色 |
| 正常        | < 120  | < 80   | 绿色 |
| 正常高值    | < 140  | < 90   | 黄色 |
| 高血压 1 级 | < 160  | < 100  | 橙色 |
| 高血压 2 级 | < 180  | < 110  | 红色 |
| 高血压 3 级 | ≥ 180  | ≥ 110  | 紫色 |

## 技术栈

| 技术         | 说明                |
| ------------ | ------------------- |
| SwiftUI      | 全量声明式 UI       |
| SwiftData    | 本地数据持久化      |
| Swift Charts | 血压 / 心率趋势图表 |
| iOS 17+      | 最低系统要求        |

## 项目结构

```
BPTracker/
├── BPTrackerApp.swift          # App 入口，注册 SwiftData 容器
├── ContentView.swift           # TabBar 主框架 + 浮动添加按钮
├── Models/
│   ├── BloodPressureRecord.swift   # 数据模型（收缩压、舒张压、心率、时间、备注）
│   └── BPLevel.swift               # 血压等级枚举（颜色、图标、判断逻辑）
├── ViewModels/
│   └── RecordStore.swift           # 数据聚合工具（分组、均值、格式化）
└── Views/
    ├── HomeView.swift          # 今日总览 + 快速记录入口
    ├── AddRecordView.swift     # 录入血压数据的表单
    ├── HistoryView.swift       # 历史记录列表（按天分组）
    ├── StatsView.swift         # 统计趋势图表
    └── SettingsView.swift      # 设置页面
```

## 运行要求

- Xcode 15+
- iOS 17+（真机或模拟器）

## 快速开始

1. 克隆仓库
2. 用 Xcode 打开 `BPTracker.xcodeproj`
3. 选择目标设备，点击运行

数据存储在设备本地，无需任何网络连接或账号。
